Projects

AWARE

AWARE, or Apple Watch Alcohol Risk Estimator, is a dual platform app, for iPhone and paired Apple Watch users, which aims to mitigate alcohol overdose and overconsumption risks through real-time user analysis and incorporation of several safety features based on a user's predicted intoxication level. The app incorporates a Machine Learning model to make real-time predictions on intoxication level based on a user's biometric data including walking steadiness and heart rate, which has never been done before in any existing solution. The app then takes several automatic actions based on the level predicted including notifying emergency services in case of a Level 3 (Severe) determination, as well as notifying / texting location to emergency contacts and enabling ride share services (using Uber API) to the user's current location in case of a Level 2 (Drunk) determination. The CS Capstone Project culminated in our work being presented in front of a panel of judges at the annual CS Summit (2024) where our team placed 2nd overall and received a cash prize of $1500.

ACE

CAAS

The CAAS (Classified Account Authorization Service) application involves managing classified accesses and handling requests for elevated privileges for government employees, interfacing between various hierarchical levels for employees to request permissions. I incorporated skillsets in the Angular CLI as well as Azure DevOps and DynamoDB to add new features including a Changelogs Page which automatically queries the database for new additions for bugfixes and updates and automatically updates the web server, as well as working on a team to help systematically upgrade the entire application from AngularJS to Angular2+ through hybridization and functionality changes.
